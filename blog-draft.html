ツール概要

人材派遣業の営業担当者向けに、提案時給が市場相場に対して競争力があるかを47都道府県別に可視化するツールを作成しました。

▶ 時給競争力シミュレーターを開く

主な機能

  • 47都道府県対応: 各都道府県の最低賃金と市場相場を表示
  • 4業界対応: 宿泊業・飲食サービス業、卸売業・小売業、生活関連サービス業・娯楽業、サービス業（派遣等）
  • リアルタイム判定: 時給を入力すると即座に全都道府県の競争力を判定
  • 4段階評価: 安全（相場+10%以上）/ 注意（相場並み）/ 危険（相場未満）/ 違反（最低賃金未満）
  • ダークモード対応

データソース

市場相場データは厚生労働省の「賃金構造基本統計調査」（e-Stat）から取得した2019年の実データを最低賃金上昇率で補正した推定値を使用しています。

  • 出典: e-Stat 賃金構造基本統計調査（厚生労働省）
  • 対象: 短時間労働者 / 1時間当たり所定内給与額 / 都道府県別
  • 基準年: 2019年（令和元年）
  • 補正方法: 2024年推定時給 = 2019年実績 × (2024年最低賃金 ÷ 2019年最低賃金)

⚠ 注意: 2020年以降、短時間労働者の産業×都道府県クロス集計データがe-Statで公開されていないため、2019年の実データを最低賃金上昇率で補正した推定値を使用しています。実際の募集条件とは異なる場合がありますので、参考値としてご活用ください。

技術スタック

  • HTML / CSS (Tailwind CSS) / Vanilla JavaScript
  • 単一HTMLファイル構成（オフライン動作可能）
  • Cloudflare Pagesでホスティング

使い方

  1. 業界を選択
  2. 提案したい時給を入力
  3. 各都道府県のカードで競争力を確認

クライアントへの時給提案時に、根拠データとしてご活用ください。
